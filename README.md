# USA Dumpsters

A modern lead-generation focused dumpster rental platform designed for fast booking workflows, responsive usability, and scalable local service expansion.

Built to create a clean and approachable customer experience while supporting long-term business growth and local SEO strategy.

## Features

- Responsive mobile-first design
- Interactive service-area mapping
- Customer-type workflow toggles
- Quote request system
- Conversion-focused landing sections
- Expandable city/service architecture
- Clean UX optimized for simple booking flow
- Local business branding integration

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Leaflet.js

## Project Goals

This project was designed to provide a scalable digital foundation for a growing dumpster rental and hauling business.

The primary focus was creating a user experience that feels approachable, trustworthy, and extremely easy to navigate across both desktop and mobile devices.

Additional goals included:

- Building a framework for future city-specific SEO pages
- Supporting future online booking integrations
- Improving customer conversion flow
- Establishing a strong regional brand presence

## Current Status

Currently in active development prior to public launch.
